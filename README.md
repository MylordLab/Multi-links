# Multi-links

Sistema web público para converter links de diversos hosts em links diretos para download.

## Visão geral
O **Multi-links** é uma ferramenta simples para quem precisa transformar links de serviços de hospedagem em **links diretos** (quando suportado), com foco em praticidade e rapidez.

> ⚠️ Este projeto **não é um serviço de host premium** e **não hospeda arquivos**. Ele apenas processa links fornecidos pelo usuário.

## Principais recursos
- Conversão de links por host (1 link por linha)
- Interface web simples e direta
- Retorno do resultado em formato “copiar e colar”


## Como usar
1. Acesse o site do projeto: https://mylordlab.github.io/Multi-links/
2. Escolha o host desejado
3. Cole os links (um por linha)
4. Execute a conversão
5. Copie os links diretos retornados (quando disponíveis)

## Limitações importantes
- A compatibilidade varia conforme o host e pode mudar sem aviso
